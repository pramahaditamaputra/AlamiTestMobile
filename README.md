# Run App on Local Machine

### Required Tools
```
NodeJS Version 14.18.2
React Native CLI
```

### Run App on Android
```
Run Command yarn to install all depedencies
Then Run Command yarn android
```

### Run App on iOS
```
Run Command yarn to install all depedencies
Then Run Command cd ios && pod install
Last Run Command yarn ios
```
